# Metriq

Metriq is a personal health analytics engine.

It aggregates nutrition, activity, and biometric data from multiple sources
and provides analytics to understand energy balance and health trends.

## Features

- MyFitnessPal nutrition ingestion
- Apple Health metric ingestion
- 7-day analytics
- Fat loss prediction

## Run locally

python main.py

Open:

http://localhost:8000/analytics
